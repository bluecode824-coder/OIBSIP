# 🚀 Oasis Infobyte Data Science Internship Projects

This repository contains all the Machine Learning and Data Science projects completed during my internship at Oasis Infobyte.  
Each project demonstrates real-world problem-solving using Python, Machine Learning, and Data Analysis techniques.

---

## 🧠 Projects Overview

### 1. Spam Message Classifier (NLP Project)
Built a machine learning model to classify SMS messages as spam or not spam using Natural Language Processing techniques such as TF-IDF and classification algorithms.

**Key Skills:** Text preprocessing, NLP, classification models

---

### 2. Sales Forecasting Model
Developed predictive models to estimate sales based on advertising expenditure across multiple media channels.

**Key Skills:** Regression analysis, feature engineering, model comparison

---

### 3. Car Price Prediction System
Created a regression model to predict used car prices based on features such as brand, year, mileage, and condition.

**Key Skills:** Data preprocessing, regression modeling, feature selection

---

### 4. Iris Species Classification
Implemented a multiclass classification model to predict iris flower species based on petal and sepal measurements.

**Key Skills:** Classification algorithms, data visualization, model evaluation
